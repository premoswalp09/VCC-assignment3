This page is part of a hybrid cloud system that demonstrates auto-scaling from a local environment to the Google Cloud Platform.

How It Works

The system monitors CPU and memory usage on the local VM. When usage exceeds 75%, the following happens:

  1. GCP infrastructure is provisioned automatically
  2. Web content is synchronized to cloud storage
  3. Load is distributed between local and cloud resources
  4. Cloud instances can scale up or down based on demand
  5. When load decreases, cloud resources are scaled back
